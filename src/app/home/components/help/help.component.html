Kanban Project Management Tool

Help

What is Join?

Join is a project management tool that provides an overview of tasks, their priorities and the responsibilities for the tasks. It uses Kanban as a method from agile project management. The goal is to ideally distribute the work to be done and the available capacities. The most important element of this method is the Kanban Board. This is divided into different work phases.

Additionally, Join offers the possibility to add new contacts under the Contacts tab.

How to use it

  1. Add a task in the To do column under the Add Task tab. Fill in all the fields to provide other team members with as much information as possible. The task will appear as a single card on the board.

  2. Assign the task to a team member or yourself.

  3. Depending on the processing status, the tasks should be moved to the appropriate columns to reflect the actual status of the workflow.
